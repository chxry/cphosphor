![Phosphor](logo.png)
## Game engine made in C.
### Features
- OpenGL deferred renderer.
- Scene editor.
- Bullet3 rigidbody physics.
- FMOD spatial audio.
- Lua scripting.
### Build Guide
1. `git clone https://github.com/chxry/phosphor --recursive`
2. `make setup`
2. `make game` / `make editor`
